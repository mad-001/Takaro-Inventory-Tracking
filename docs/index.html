≡ƒôï Prerequisites

  • Node.js (v14 or higher)

≡ƒÜÇ Quick Start

  1. Download and extract the latest release
  2. Run npm install to install dependencies
  3. Copy config.example.json to config.json
  4. Edit config.json with your Takaro domain name
  5. Start the server:
    • Windows: Double-click start-server.bat
    • Linux/Mac: Run node server.js
  6. Open your browser to http://localhost:5555

≡ƒôû How to Use

  1. Login with your Takaro credentials
  2. Select a game server from the dropdown
  3. Enter search parameters:
    • X and Z coordinates (center of search area)
    • Radius in game units
    • Time range to search
  4. Click "Search Inventory"
  5. Analyze the results:
    • Click column headers to sort
    • Use the filter to search for players or items
    • Green badges = items added (+)
    • Orange badges = items removed (-)

≡ƒöº Troubleshooting

Server won't start

  • Make sure config.json exists (copy from config.example.json)
  • Check that port 5555 isn't already in use
  • Verify Node.js is installed: node --version

Can't login

  • Verify your Takaro email and password are correct
  • Check that your domain name is correct in config.json
  • Ensure you have API access enabled in Takaro

No results found

  • Verify coordinates are correct (check in-game position)
  • Try increasing the search radius
  • Check the time range includes when activity occurred
